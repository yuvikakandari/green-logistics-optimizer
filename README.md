# Green Logistics Optimiser

An intelligent route optimisation system that combines:

- Dijkstra's Algorithm
- XGBoost Travel Time Prediction
- FastAPI Backend
- Streamlit Dashboard
- Folium Maps

## Features

- Shortest path calculation
- Traffic-aware route optimisation
- Travel time prediction
- Interactive route visualisation

## Tech Stack

Python, NetworkX, XGBoost, FastAPI, Streamlit, Folium
